Energy-Efficient Edge AI

Designing low-power deep learning architectures, memory-optimized DNN accelerators, and cross-layer model compression techniques that achieve up to 94% energy-delay improvements — enabling AI inference on resource-constrained edge devices.

DNN Accelerator Design DRAM Energy Optimization Pruning & Quantization HW-Aware NAS Approximate Computing Tiny ML

ML Security & Privacy

Comprehensive adversarial robustness research spanning attack detection, backdoor defense, deepfake identification, LLM jailbreak analysis, and side-channel mitigation — building trustworthy AI systems resilient to emerging threat vectors.

Adversarial Attack Defense Backdoor Detection Deepfake Analysis LLM Jailbreak Research Side-Channel Mitigation Hardware Obfuscation

Neuromorphic Computing

Spiking neural network (SNN) research leveraging neuromorphic processors for ultra-low-power inference — with demonstrated applications in autonomous driving, smart agriculture, and event-driven vision using dynamic vision sensors.

Spiking Neural Networks Neuromorphic Processors Event-Driven Vision Continual Learning Smart Agriculture ADAS Applications

Embodied AI & Robotics

Brain-computer interface (BCI) controlled robotic arms using EEG and gaze tracking, autonomous rover systems with SLAM-based exploration, and real-time object search using vision-language models with sub-20cm localization accuracy.

EEG-Gaze Robotic Control SLAM & Exploration Vision-Language Navigation Trajectory Planning Real-Time Object Search Sensor Fusion

Edge AI for Healthcare

FPGA-accelerated medical image processing for real-time ultrasound analysis, automated brain MRI lesion segmentation with multimodal LLM-generated diagnostic reports, and energy-efficient hardware for deployable clinical edge systems.

Medical Image Segmentation Ultrasound Edge Processing FPGA Accelerators MLLM Diagnostic Reports Fetal Feature Detection Prognostic AI

Quantum Machine Learning

Design-space exploration of hybrid quantum-classical neural network architectures, quantum federated learning for privacy-preserving AI, adversarially robust quantum models, and real-world QML applications in climate modeling, cybersecurity, and portfolio optimization.

Hybrid QNN Architectures Quantum Federated Learning Barren Plateau Mitigation Circuit Cutting Quantum Code Generation Portfolio Optimization

Multimodal LLM Research

Agentic LLM systems for offensive security (capture-the-flag), AI-generated image classification, healthcare multimodal models, hardware code generation, LLM-driven quantum circuit synthesis, and high-accuracy model cascading for code completion.

Agentic Security Systems AIGC Detection Healthcare MLLMs RTL Code Generation Digital Avatars Code Cascading

Autonomous Application Security Platform

Agentic AI · AppSec · DevSecOps

A multi-agent security platform that autonomously discovers, tests, and remediates access-control vulnerabilities in AI-built applications. A vision agent maps every workflow, endpoint, and role; a security agent executes thousands of attack simulations across tenants and permission boundaries; and a fix agent generates verified patches through popular code agents — all from a single URL input with zero configuration.

Venture Partner Role

Product Strategy — Guided core architecture decisions and user experience design to establish category leadership in autonomous application security
Go-to-Market — Defined market positioning and competitive differentiation to capture a whitespace opportunity between legacy security tooling and the AI-native development wave
Adoption & Scale — Engineered the product-led growth motion that accelerated enterprise and startup adoption through frictionless onboarding and rapid time-to-value
Multi-Agent Architecture Autonomous Pentesting Access Control Analysis Multi-Tenant Isolation Auto-Remediation

Agentic AI Memory Infrastructure

Memory Layer · RAG · Agent Orchestration

A self-hostable memory infrastructure platform that gives AI agents persistent, searchable memory that survives context windows. Connects knowledge bases, documents, and interaction history to LLM context — enabling RAG systems, personalized AI assistants, and complex multi-agent workflows with shared memory and coordination. Fully provider-agnostic, running across cloud, on-prem, and local hardware.

Venture Partner Role

Product Strategy — Shaped the platform architecture and vendor-agnostic deployment model to maximize enterprise appeal and eliminate lock-in concerns
Go-to-Market — Positioned the platform as foundational infrastructure for the agentic AI stack, targeting developer teams building next-generation RAG and multi-agent systems
Adoption & Scale — Designed the developer-first growth strategy that drives organic adoption through low-friction onboarding and broad ecosystem compatibility
Persistent Agent Memory Semantic Search RAG Infrastructure Multi-Agent Coordination Provider Agnostic
