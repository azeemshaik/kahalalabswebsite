# Kahala Labs — Official Website

Corporate website for **[Kahala Labs](https://kahalalabs.com)**, a technology research and consulting firm that bridges academic AI research with enterprise commercialization. The site covers three core business pillars — Research, Ventures, and Consulting — and presents Kahala Labs' global presence across the US, UAE, and India.

---

## Getting Started

**Prerequisites:** Node.js 18+, npm

```bash
# Install dependencies
npm install

# Start dev server with HMR (http://localhost:5173)
npm run dev

# Type-check + production build → dist/
npm run build

# Preview the production build locally
npm run preview

# Lint
npm run lint
```

### Research domains (7 cards)

1. **Energy-Efficient Edge AI** — DNN accelerators, pruning, quantization, TinyML
2. **ML Security & Privacy** — adversarial defense, backdoor detection, deepfake, LLM jailbreak, side-channel
3. **Neuromorphic Computing** — SNNs, neuromorphic processors, event-driven vision
4. **Embodied AI & Robotics** — BCI/EEG robotic control, SLAM, vision-language navigation
5. **Edge AI for Healthcare** — FPGA accelerators, medical image segmentation, MLLM diagnostic reports
6. **Quantum Machine Learning** — hybrid QNN, federated QML, circuit cutting, portfolio optimization
7. **Multimodal LLM Research** — agentic security, AIGC detection, RTL code generation, code cascading

### Ventures sub-sections

- **Portfolio Spotlights** — Autonomous Application Security Platform; Agentic AI Memory Infrastructure
- **AI Infrastructure Site Portfolio** — 3 GPU-dense data center sites (Site Alpha/Beta in South Asia, Site Gamma in Southeast Asia); 100–200 MW phased capacity; liquid-cooled, high-density racks
- **Data Center Infrastructure Brokerage** — optical fiber cable + industrial power cable brokerage for US data center market
